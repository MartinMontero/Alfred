Alfred beta 3 — the first update delivered through Alfred itself.

What changed:
- Create a vault from anywhere: both vault menus and the new Vaults entry in the sidebar can make a new vault, not just open existing folders.
- Sync says what it is: your notes sync over your own Nostr relays — the cloud icon and "cloud sync" wording are gone.
- The agent is one click away: the "agent idle" status now opens the agent panel, and every icon in the app has a name you can read by hovering.
- The last of the old purple is gone from the onboarding art and the graph.

Updates arrive inside the app: Settings → About → Check for updates.
Known issues: https://github.com/MartinMontero/Alfred/blob/main/docs/beta/known-issues.md
